博客网站搭建

2018-04-02

GitHub Pages

Hosted directly from your GitHub repository. Just edit, push, and your changes are live.

github提供的构建个人网站的方案。给你提供服务器存放文件，甚至给你提供一个免费的域名，简直太香。官网链接如下 https://pages.github.com/

Hexo

A fast, simple & powerful blog framework

  • 环境准备：
    1. nodejs 官网：https://nodejs.org/en/

比较好用的一点是，hexo支持主题，用户可以设计自己的博客网站外观，可以开源出来给其他人使用修改，比如我的博客网站主题来自：https://github.com/esappear/hexo-theme-clover

具体可以参考hexo的网站说明 https://hexo.io/

Markdown

有了博客网站，博客文章如何编写呢？当然你可以用word编写，用各种文档编辑软件来写，但作为程序员，肯定是用【语言】来写文章更熟悉一点。

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。Markdown能被使用来撰写电子书，如：Gitbook。

比如本篇博客用到如下标记：

  • ### 三级标题
  • - 无序列表

入门教程可参考这里 https://www.runoob.com/markdown/md-tutorial.html
