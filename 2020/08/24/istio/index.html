istio四个核心功能

  • 连接

智能控制服务之间的流量和 API 调用，进行一系列测试，并通过红/黑部署逐步升级。

  • 保护

通过托管身份验证、授权和服务之间通信加密自动保护您的服务。

  • 控制

应用策略并确保其执行，使得资源在消费者之间公平分配。

  • 观测

对您的一切服务进行多样化、自动化的追踪、监控以及记录日志，以便实时了解正在发生的事情。
